# mandril-quotes

Package for getting quotes from the Danish TV show "Casper & Mandrilaftalen"

## installation

`npm i mandril-quotes`

## usage

```JavaScript
const mandrilQuotes = require("mandril-quotes");

mandrilQuotes.spisIld();
```
